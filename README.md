# 🏥 [Nome do Projeto] - Sistema de Consultas Médicas

Projeto Integrador de Módulo (4º Período) - Análise e Desenvolvimento de Sistemas.

## 🎯 Objetivo
Desenvolver uma Solução Web Full-Stack para agendamento e gerenciamento de consultas médicas, utilizando mensageria para processamento assíncrono.

## 🛠 Tecnologias Utilizadas
* **Frontend:** React (em breve)
* **Backend:** Spring Boot (Java)
* **Mensageria:** RabbitMQ (ou Kafka)
* **Design e Prototipagem:** Figma

## 📂 Estrutura do Repositório
* `/backend`: API RESTful e lógica de negócios.
* `/frontend`: Aplicação Web de interface.
* `/docs`: Documentação técnica e arquitetural.
